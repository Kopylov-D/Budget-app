# Budget App
Демонстрационная версия - [Budget App](https://budget-app-k.web.app/)

## Используется:

* ReactJS
* Redux
* Redux-thunk
* React-router
* Axios

## Реализовано:

* Регистрация и авторизация
* Хранение записей в базе данных на сервере
* Валидация ввода email и пароля
* Маршрутизация
